About

Welcome to the Keosha Area Transit() mobile application. The purpose of the application is to provide navigation around Kenosha using the Kenosha buses. By using routing (by entering the start and end addresses), you can get a detailed description of how to get from the start to the end using Kenosha buses.

By using black and white QR codes (found on the bus signs and scanned by the application), you can get a list of the routes, times, and directions that are coming to that stop.

Fares:

  • Adult – 1.75
  • Student – 1.25
  • Elderly and Disabled – 0.85

Fares include one free transfer (request upon boarding)
