Welcome to the Kenosha Area Transit () mobile application

The purpose of the application is to provide navigation around Kenosha, using the Kenosha buses.

By using routing (with you entering the start and end addresses), you can get a detailed description of how to get from the start to the end using Kenosha buses

By using black and white QR codes (found on bus stop signs and scanned by the application), you can get a list of the routes, times and directions that are coming to that stop.

Fares: Adult – 1.75, Student – 1.25, Elderly and Disabled – 0.85

Fares include one free transfer (request upon boarding)

How To

How to find out what buses are coming to your stop

  1. Press the Back button
  2. Press the Scan button
  3. Hold the phone up to the bus stop sign so the black and white QR code is visible on the phone display
  4. When the phone automatically captures a picture of the QR code, wait for the application to process it
  5. Now visible on the screen will be the route numbers, arrival times and directions which describe the coming buses to that stop

How to get directions from one location to another

  1. Ensure that there is an internet connection on the phone
  2. Press the Back button
  3. Press the Route button
  4. Enter a start address and an end address in the appropriate spaces
  5. Press submit
  6. Now visible on the screen will be the detailed description of how to travel from the start to the end using Kenosha buses

How to change the language of the application

  1. Press the Accessibility button
  2. Press the Espanol or English button to change the language of the words in the application
  3. All of the text in the application is now displayed in the language chosen

How to set up voice feedback

  1. Press the Accessibility button
  2. Press the Text to Speech button
  3. All of the text in the application is now spoken instead of only in text form

How to make the text larger (zoom)

  1. On any application page, place two fingers on the phone touch screen
  2. To make text larger, slide the two fingers farther apart from each other while keeping them on the surface of the screen
  3. To make text smaller, slide the two fingers closer together while keeping them on the surface of the screen

How to reach the Kenosha Area Transit website

  1. Ensure that there is an internet connection on the phone
  2. Press the Back button
  3. Press the banner at the top of the touch screen

Accessibility

  • Text to Speech button or checkbox (Press here to allow the application to read text content aloud)
  • Espanol button or checkbox (Press here to allow the application to display the text to Spanish)
  • English button or checkbox (Press here to allow the application to display the text to Spanish)
