Accessibility

How to change the language of the application

    For iPhone
  1. Click on the Settings icon on your phone main screen.
  2. Scroll down to General and click it
  3. Find the section that says International and select it
  4. Click the section that says Language and select either Spanish or English
  5. The app will now display the language that your phone is set to. If your phone is using a language that is not Spanish or Enlgish it will default to English.

How to set up voice feedback

    For iPhone
  1. Click on the Settings icon on your phone main screen.
  2. Scroll down to General and click it
  3. Find the section that says Accessibility and select it
  4. Select the section that says Speak selection
  5. On this page switch the speak selection switch so that it is on (the background for the switch being on should be green)
  6. Now when you highlight a section, all of the text in the application is now spoken instead of only in text form
